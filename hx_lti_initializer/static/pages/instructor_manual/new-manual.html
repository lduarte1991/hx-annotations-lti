HxAT Instructor Manual

The following manual will guide the instructor through setting up the HxAT in edX. It will also go through the steps of setting up the tool in the Admin Hub. A PDF version of this manual is also available.

1. Adding LTI module to edX

First thing you need to do is head over to the "Advanced Settings" page in Studio.

1.1 Advanced Settings - Advanced Module List

Add "lti_consumer" (all lowercase!) to the "Advanced Module List".

1.2 Advanced Settings - LTI Passports

Now scroll down to the LTI Passports option. Here you will have to input an ID (could be anything you like, I call it "hxat" or "hx_annotations") followed by a colon, followed by the key for the LTI tool (ask tech for this information) and the secret password for the LTI tool (also ask tech for this) after a second colon.

1.3 Save

Once you save, studio scrolls you all the way to the top. Check to ensure the information you added was saved correctly.

2. Setting Up HxAT in LTI Module

Now head over to the section in the course where you want the annotation tool to appear. Once you've created a new unit, you will see an "Advanced" button as an option to add a new component.

2.1 Add LTI Module

Now click on the "LTI Consumer" module option.

2.2 Editing the LTI Settings

Now you will see a generic component for LTIs we will go into edit mode to set up the appropriate settings.

2.3 Settings - Part 1

LTI should be the first part of what you put into the LTI Passport.
Change the LTI URL to the one below https://hxat.harvardx.harvard.edu/lti_init/launch_lti/.
There are other options like opening the tool in a new page or making the tool send back a grade (participation grade online), but for the most part you can leave this untouched.

2.4 Settings - Part 2
